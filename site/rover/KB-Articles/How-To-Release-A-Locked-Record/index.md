# How to release a locked record

<PageHeader />

### Unlocking a Locked Record

1. Navigate to **System.D**. (Only admin users may have access to this command) **Note:** If you do not have access to this command check with your system administrator

2. Open **User List**.

3. Locate the user who is locking the record.

4. Check with the user and ask them to log out. If they are in the middle of processing data, you can end up mismatched/corrupt data.  The logoff command should only be used when a line is truly hung.

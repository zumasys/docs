# Unable to create the excel file (Legacy Client)

<PageHeader />
1. Determine Scope
Check if the issue is occurring for a single user or all users.
2. Verify Excel Application
Confirm whether Excel is listed as an application in SECURITY.E2.
3. Enable "Use XLS"
In SECURITY.E2, ask users to check the "Use XLS" box.

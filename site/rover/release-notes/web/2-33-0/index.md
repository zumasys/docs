# Rover Web v2.33.0 Release Notes

<badge text= "Version 2.33.0" vertical="middle" />

<PageHeader />

These are the release notes for version 2.33.0 of the Rover Web application and can be made available to customers running _Rover ERP_, _IMACS_ and other non-Zumasys owned systems. Contact your Zumasys account representative for more information about upgrading.

## New Features

### General

- Improved responsive behavior across Rover Web for a more consistent experience on different screen sizes.
- Improved dialog and form layouts in several areas of the application for better readability and interaction.
- Added support for per value custom styling on lookup driven datatables.

### Customer Inquiry

- Added support for lookup-based invoice display in **Customer Inquiry**, improving flexibility for invoice selection and display.
- Improved invoice actions in Customer Inquiry, making it easier to:
  - download invoices
  - email invoices
  - add or remove invoices for payment workflows

### Point of Sale

- Improved offline search to support injected logic that applies order header selection as criteria in part selection.
> Only supported by select ERPs

## Bug Fixes

### Point of Sale

- Fixed several responsive layout issues in **Point of Sale** screens, including customer, order, payment, and receipt history views.

### Scan 

- Improved scan screen usability with more consistent input sizing, dialog behavior, and focus handling.

### Customer Inquiry

- Improved Customer Inquiry invoice handling so lookup-driven invoice results display more reliably.
- Improved invoice count and badge updates when working with invoice data in Customer Inquiry.
- Fixed responsive layout and interaction issues in sales-related screens, including orders, opportunities, and supporting dialogs.

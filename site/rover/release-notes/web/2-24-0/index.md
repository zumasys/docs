# Rover Web v2.24.0 Release Notes

<badge text= "Version 2.24.0" vertical="middle" />

<PageHeader />

These are the release notes for version 2.24.0 (3/27/2026) of the Rover Web application and can be made available to customers running _Rover ERP_, _IMACS_ and other non-Zumasys owned systems. Contact your _Client Success Manager_, [Sales](mailto:sales@zumasys.com?subject=Rover%20Web%20v2.24.0) or [Support](mailto:help@zumasys.com?subject=Rover%20Web%20v2.24.0) today!

## New Features

### Scan

 - The font for text and placeholders in input fields for Scan apps was changed from fixed-width to standard font used elsewhere.
 - A new user setting, Select on Focus, has been added to the Settings page. When enabled (default), clicking into an input field on Scan forms will highlight the full text, allowing for easier input, particularly for numbers.

## Bug Fixes

### Point of Sale

- Corrected a change in behavior after receipt printing causing user to be returned to customer selection unexpectedly.

### Scan

- For Formsdef-defined Scan forms, the first editable field of the form is now activated on opening the form.

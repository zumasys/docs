These are the release notes for version 2.25.1 (04/09/2026) of the Rover Web application and can be made available to customers running _Rover ERP_, _IMACS_ and other non-Zumasys owned systems. Contact your _Client Success Manager_, [Sales](mailto:sales@zumasys.com?subject=Rover%20Web%20v2.25.1) or [Support](mailto:help@zumasys.com?subject=Rover%20Web%20v2.25.1) today!

## Bug Fixes

### Rover Web

#### Point of Sale

- Adjustments were made to handling of scan part # input to allow for concatenated part values to be handled properly when separated by a defined scanner suffix.

### Rover Portal

#### General

- Additional details were added to API integration layer for host based validation logic.

#### Invoices

- Addresses payment errors not being reported to the user properly.

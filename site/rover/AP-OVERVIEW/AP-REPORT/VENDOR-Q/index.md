**Purpose**  
The VENDOR.Q procedure provides an on-line inquiry for a selected vendor. Included at the bottom of the screen is an aging of all balances according to the aging periods defined in the [ AP.CONTROL ](../../AP-ENTRY/AP-CONTROL/index.md) file.

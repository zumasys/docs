=======================================================================================================
ROVER ERP - COMPLETE COMMAND/FORM REFERENCE GUIDE
=======================================================================================================

This is a quick reference for finding documentation on specific Rover ERP commands, forms, and processes.

Forms are organized by module using this naming convention:
  [MODULE]-E       = Data Entry Form (e.g., PO-E, SO-E, LABOR-E)
  [MODULE]-P1      = Process/Batch (e.g., MRP-P1, CYCLE-P1)
  [MODULE]-R1      = Report (e.g., PO-R1, AR-R1)
  [MODULE]-Q       = Inquiry/Query (e.g., EXEC-Q, INV-Q)
  [MODULE]-F1      = Form Report (e.g., PO-F1, PHYS-F1)

=======================================================================================================
HOW TO USE THIS CATALOG
=======================================================================================================

1. Find your task or module in the "COMMON TASK LOOKUP" section above
2. Note the location pattern: [MODULE]-[TYPE]/[FORM]/[STEP]/README.md
   Example: MRK-OVERVIEW/MRK-ENTRY/SO-E/SO-E-1/README.md

3. Full path would be:
   C:\source\internal\docs\site\rover\MRK-OVERVIEW\MRK-ENTRY\SO-E\SO-E-1\README.md

4. Each README.md contains:
   - Step-by-step instructions
   - Field descriptions
   - Screenshots (.jpg files in the same folder)
   - Related processes
   - Links to other forms

5. Each form typically has 1-7 numbered steps (e.g., SO-E-1, SO-E-2, ..., SO-E-9)
